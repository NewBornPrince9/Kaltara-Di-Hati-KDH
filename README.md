# Kaltara-Di-Hati-KDH
**Kaltara Di Hati (KDH)** adalah inisiatif aset digital berbasis blockchain (*Binance Smart Chain*) yang dirancang khusus untuk mempercepat transformasi ekonomi digital di Provinsi Kalimantan Utara. Proyek ini menggabungkan teknologi keuangan modern dengan semangat konektivitas lokal.
